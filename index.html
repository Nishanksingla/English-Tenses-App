IDENTITY

जिन वाक्यो का अन्त ता है, ते है, ति है से होता है।

Helping Verb:

Verb:

Note: In this tense 'Since' and 'For' is used along with time.
'For' is used when we measure the duration - when we say how long something lasts, e.g for 2 hrs.
'Since' is used when we have the starting point of actions, events or states. It refers to when things began, e.g Since morning.

Sentence Formation Rules

  • Positive Sentence

    Sub + had + been + V1+'ing' + obj + since/for + time.

  • Negative Sentence

    Sub + had + not been + V1+'ing' + obj + since/for + time.

  • Interogative Sentence

    Had + sub + been + V1+'ing' + obj + since/for + time ?

Future Perfect Continuous

Future Perfect Contiuous Tense

Type of Forms

  • Positive Form

    Sub + will/shall + have + been + V1+'ing' + obj + since/for + time.

  • Negative Form

    Sub + will/shall + not + have + been + V1+'ing' + obj + since/for + time.

  • Interogative Form

    Will/Shall + sub + have + been + V1+'ing' + obj + since/for + time ?

Rules

  1. 'Will have been' and 'Shall have been' are the Helping Verbs.
  2. We use 'For' when we measure the duration - when we say how long something lasts, e.g for 2 hrs.
  3. We use 'Since' when we have the starting point of actions, events or states. It refers to when things began, e.g Since morning.

When to use this tense

जिन वाक्यो का अन्त ता है, ते है, ति है से होता है।
